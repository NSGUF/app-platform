# Vite+vue2项目模板

## 介绍
集成Vite、vue2、vueRouter、Vuex、sass项目开发模板

注意：
- 由于vue官网脚手架[vue-cli](https://cli.vuejs.org/zh/index.html)内置的构建工具webpack打包太慢，所以改为vite工具。
- [vite](https://vitejs.cn/)官网脚手架初始化的项目是基于vue3，我们需要改为vue2。

## 使用

1.进入项目目录，安装依赖，运行项目

```
npm install
npm run dev
```

2.打包项目,预览项目

```
npm run build
npm run preview
```
